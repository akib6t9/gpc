Ghoshkanda Pre-Cadet and High School - Modern Premium Template
-------------------------------------------------------

This is a GitHub Pages-ready static website template (HTML/CSS/JS) for Ghoshkanda Pre-Cadet and High School.

How to use:
1. Download the zip and unzip.
2. Upload the files to a GitHub repository (public).
3. In the repository settings enable GitHub Pages from the main branch.
4. Your site will be available at: https://{your-github-username}.github.io/{repo-name}/

Files included:
- index.html, about.html, teachers.html, admission.html, contact.html
- assets/ (logo.png, style.css, teacher images)
